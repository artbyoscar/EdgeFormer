kv_cache_offload

Enable KV cache offloading to disk for a model.

Args: model: The EdgeFormer model to enable offloading for offload_path: Path to store KV cache files (default: temporary directory) kv_cache_dtype: Data type for KV cache (default: same as model)

Returns: Model with KV cache offloading enabled
